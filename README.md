# React Router Middleware on Vercel demo

A simple demonstration for using React Router v7+ middleware on Vercel platform.

## Setup

1. Put `server/app.ts` to pass `RouterContextProvider`.
2. Add `build` option to use the `server/app.ts` in `vite.config.ts`.
3. Use middleware
